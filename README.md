# FC Friendlies Tracker

A comprehensive tracking and tournament management system for EA FC 26 online friendlies on PS5.

## Features

- **League Table**: Track standings with H2H tiebreaker and goal average
- **ELO Ratings**: Dynamic ratings (K=32) updated after each match
- **Head-to-Head Matrix**: Visual W-D-L records between all players
- **Match History**: Complete log of all matches played
- **Tournament Mode**:
  - Knockout bracket with auto-progression
  - Group stage with fixtures generator
  - Champions League style pot-based draw
- **700+ Teams**: Full FC 26 team database with searchable selector
- **OCR Score Scanning**: Twitch frame capture or screenshot upload

## Tech Stack

- Next.js 14 (App Router)
- React 18
- localStorage for persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
